How to Convert Column to Semicolon Separated List

  1. Copy your column data. (Example: Excel column data)
  2. Paste the column on the left or type data in columns.
  3. It automatically convert any column data to Semicolon separated list.
  4. Copy your Semicolon separated list on the right.
  5. Use or Paste your Semicolon separated values anywhere without any issues.

Pro tip: Click on settings tab to change delimiter value. Semicolon separator online tool also supports prefix and suffix values in the both input column data and delimited character results. Enter your own delimiter character to get delimited list values.
