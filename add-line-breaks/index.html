Add line breaks easily to split text into multiple lines. You can select where to add a line break like after specific character/word or an occurrence of a letter/word. You can also add line breaks by selecting “Add 1 line” option to double the line breaks. It has many features including upload your word document, add line breaks then save your word file.

How to Use Add Line Breaks Online tool

To use our add line breaks online tool follow these steps:

  1. Enter your text or upload a word file.
  2. Then Select Add line Breaks (after, before and instead of) options and enter your choice of word/character.
  3. Now, select text occurrence or character position or add 1 line option.
  4. Then select remove one or replace with or remove all line breaks.
  5. Now, click on apply button. Use clear option to start again.
  6. Optional: Download or Save your output as word document.

Add Line Breaks – Text Occurrence

This is most useful option that can add line breaks before or after certain characters. Simply select before or after option and enter your character/word to use this option.

You can also choose “instead of” option to easily replace the specified character with a new line.

Example – Add line breaks after each occurrence of letter “C”.

ABCABCABC

Result:

ABC
ABC
ABC

Add Line Breaks – Character Position

This is another feature that you can use to add a line break after specific number of characters.

Example: Add line breaks every 4 characters.

ABCABCABC

Result:

ABCA
BCAB
C

Add Line Breaks – Add 1 Line

This is handy option to add a new line after the last occurrence of line break followed by any character except \n.

Example: Add new line

ABC
ABC
ABC

Result:

ABC

ABC

ABC
