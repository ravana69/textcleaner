Morse Code allows operators to send messages using dots and dashes to communicate during difficult and noisy environments. Morse code was invented by Samuel Morse in 1840s. It was widely used in amateur radio, military and aviation activities.

In Morse code, longer signals represented by dash and shorter signals represented by dots. In the past, it is a highly reliable and effective communications method in the military and aviation.

Samuel Morse was sent “What hath God wrought” message in Morse code to open the Washington telegraph line on May 24, 1844. He got U.S patent for Morse code (dot-dash telegraphy signals) on June 20, 1840.
