What is Invisible character

An invisible character is nothing but blank space, white space or empty character are actually unicode characters. These invisible characters are commonly used to send empty messages on WhatsApp, Instagram, Facebook, Twitter, pubg, fortnite and many sites. Invisible character does not mean empty space, it is character.

How to copy paste invisible character

  1. Click on copy button to copy paste invisible character.
  2. You might want multiple invisible characters, then enter number of empty characters and click on generate button.
  3. Click on copy button or send empty message on WhatsApp by clicking on WhatsApp button.

That’s it!
